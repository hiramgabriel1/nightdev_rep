# NightDev Bot

NightDev is a Telegram bot to manage repositories, deploy keys, and configs directly from chat.

## Features

- Manage repos via Telegram
- Deploy keys management
- Config viewer and editor
- Prisma + SQLite backend

## Getting Started

1. Clone the repo
2. Run `yarn install`
3. Configure your bot token
4. Start with `yarn dev`

## Tech Stack

- **Runtime:** Node.js
- **Framework:** Telegraf
- **Database:** SQLite + Prisma CLI
- **Git:** SSH-based operations
